# Template for Cisco Catalyst 9500 switches

This template covers basic SNMP for Catalyst 9500 Series but it may also work for other Catalyst 9XXX Switches.

Among the things included in the template :

- CPU usage (5 seconds, 1 and 5 minutes)
- Network interfaces (Status, Alias, Description, Traffic, Name, MTU, etc)
- RAM usage (Free, Used and Total RAM)
- Temperature sensors (Description, State, Temperature and Threshold)
