# Template for HP t520 thin client

This template is for monitoring some basic hardware values from HP t520 thin clients on Linux.


## Includes queries

With this template you can query:
- CPU temperature
- GPU temperature
- Power usage


## Triggers

To get notified there are triggers configured for
- High CPU / GPU temperature
- High power usage


## Requirements

- Linux OS
- Package "lm-sensors" needs to be installed if missing
